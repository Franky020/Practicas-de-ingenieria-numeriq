# Prácticas de Ingeniería

Este repositorio contiene el desarrollo de las **prácticas anteriores** realizadas durante la materia, incluyendo tanto el **frontend** como el **backend** del proyecto.

---

## Estructura del proyecto

El repositorio está organizado de la siguiente manera:

/frontend → Aplicación desarrollada en React
/backend → API desarrollada en Laravel


---

## Tecnologías utilizadas

### Frontend
- React
- Vite
- JavaScript / TypeScript
- HTML / CSS

### Backend
- Laravel
- PHP
- API REST

---

## ⚙️ Instalación y ejecución

### 🔹 Clonar repositorio
```bash
git clone https://github.com/tu-usuario/tu-repo.git

cd frontend
npm install
npm run dev

cd ..

cd backend
composer install
php artisan serve
```
Notas
Este repositorio tiene fines académicos.
El frontend consume la API del backend para pruebas de funcionalidad.
